Light Sharp Corporation: Website Design in Webflow

Overview

This was a freelance gig for Light Sharp Corporation. I took the mid-fidelity desktop design and built it out responsivey using Webflow, ensuring that the code was produced correctly and adding polish to the design.

Context and Challenge

Background and Description

The client wanted me to build an easy-to-manage responsive website using Webflow. Webflow is a SaaS program that allows users to build out their website design, and it produces the corresponding code for you based on how you design. This allows clients to easily update their websites as needed in the future without having to take a deep-dive into full-stack web development.

Problem

I had never before used Webflow, however I knew design, HTML, and CSS, so I figured this would be doable.

Goals

Build out the mid-fidelity design provided into the program Webflow. I needed to consider all screen sizes from mobile landscape and portrait to large desktop, and I needed to do this in such a way that the code I produced was clean, manageable, easy to interpret, and concise.

Process and Insights

I spent some time on Webflow University watching all of the introductory instructional videos until I felt like I had a basic grasp on how things worked here in Webflow. I got started on building out the design provided me (pictured above), and when I stumbled, I simply went back to Webflow University to find an in-depth video instructing me on the topic at hand. I learned some neat things along the way, like how to embed a Google Maps widget into Weblow, and how to get the forms to collect user input correctly, as well as send feedback to the users based on their input. After things were all in their proper places and working correctly, I added finesse to the design.
